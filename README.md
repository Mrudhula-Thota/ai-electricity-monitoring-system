# AI Electricity Monitoring System

## Project Description
This project monitors electricity usage and predicts future consumption using historical data.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Features
- Login System
- Dashboard
- Electricity Usage Chart
- Data Storage
- AI Prediction
- Logout System

## How to Run
1. Install XAMPP
2. Place project in htdocs
3. Start Apache and MySQL
4. Open localhost/ai-electricity-system/login.html
